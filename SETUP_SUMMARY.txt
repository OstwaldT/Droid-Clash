================================================================================
🎮 PROJECT STRUCTURE
================================================================================

Your game is a **multiplayer browser-based party game** featuring:

  • Architecture: Godot 4.2 server ↔ Vue 3 browser clients (WebSocket)
  • Gameplay: Turn-based robot/tank battles on hexagonal grid
  • Players: 2-8 players connecting via mobile browsers
  • Cards: Players select 3 instruction cards per turn
  • Tech Stack: GDScript (server), Vue 3 + Tailwind (client)
